# Mode3 Website

A modern, responsive front-end website for **Mode3**, an ICT/IT consulting and services company.

## Overview
This project is a static marketing website with:
- Hero, About, Services, Capabilities, Insights, and Contact sections
- Dark mode toggle
- Mobile navigation
- Scroll reveal animations
- Partner logo placeholders and client reviews section

## Languages Used
- **HTML5** (`index.html`)
- **CSS3** (`styles.css`)
- **JavaScript (ES6+)** (`script.js`)

## Running Locally
```bash
python3 -m http.server 4173
```
Then open:
- `http://127.0.0.1:4173`

## Project Structure
- `index.html` — page structure and content
- `styles.css` — styling, layout, responsive behavior, theme variables
- `script.js` — interactions (theme toggle, mobile menu, scroll effects, form handling)

## Notes
- Partner logos are currently placeholders and can be swapped with official assets.
